# 🚗 Eco Clean Car Wash Website

A modern and responsive Eco Clean Car Wash website developed using HTML, CSS, and JavaScript. This project provides users with an easy-to-use interface to explore car wash services, book appointments, and learn about eco-friendly cleaning solutions.

---

## 📌 Project Overview

The Eco Clean Car Wash Website is designed to promote environmentally friendly car washing services. It offers a clean, responsive, and user-friendly interface that works smoothly across desktop and mobile devices.

---

## ✨ Features

- 🏠 Responsive Home Page
- 🚗 Car Wash Service Details
- 📅 Online Booking Form
- 💰 Service Plans & Pricing
- ⭐ Customer Reviews Section
- 📞 Contact Information
- 📱 Mobile-Friendly Design
- 🎨 Modern and Attractive User Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
